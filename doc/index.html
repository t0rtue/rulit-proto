The list of games

Public and Local

Public games are visible by everyone. Local games are your own games, stored on your computer.

Create a game (1)

You can create as many Local games as you want. To do so you have to enter a name and then click on "Create"
You can not yet share your Local game and make it Public.

Fork a Public game

If you edit and save a Public game, this will create a new Local game with the same name and rules.

Play a game (2)

Mouse over a game and click on the "play" icon to access the Game view.

View and edit rules (2)

Mouse over a game and click on the "gears" icon to access the Rulebook.

The Game view

Navigation

At the top of the page, the navigation menu allows to switch between the Rulebook and the Game menu . You can also go back to the Games List by clicking on "Rulit" at the right.
At any moment during the game, if you click on then the game ends and you return to the game menu.

Game Menu

The entry point of a game, displays some information according to defined rules: name, description and a reminder of the end conditions.

Players management

You can add and remove players according to the limits (min and max) set by the rules. You can modify the name of each player, limited to 8 characters.

Launch game

To launch the game, just click on the big green play button.

Game Interface

You can play the game, according to rules. All the necessary elements are present:
  • Player informations
  • Turn informations and controls
  • Board
  • Allowed actions
  • Message box

The Rulebook

Navigation

At the top of the page, the navigation menu allows to switch between the Rulebook and the Game menu . You can also go back to the Games List by clicking on "Rulit" at the right.

Save

To save your modifications you can click on the save icon : /
The game will be locally saved. If it is a Public game then a fork will be done and locally saved.

View and edit

The rulebook is fully editable, allowing to define the rules of your game or make variants of an existing game.
You can navigate in the rulebook using the left menu. There are 3 main parts:
  • The effective game rules
  • Help and game tips
  • Style and theme of the game

Things to try

  • Edit one of the demo games and change the board type: square / hexa. Then launch the game and experiment the various actions (put, move) on this new grid.
  • Edit Yavalath game and change the board type and size. You have now a new game with maybe interesting strategies.
  • Edit Yavalath game and change the win and lose conditions. You have now a new game with maybe interesting strategies.
  • Edit Yavalath game and add a phase during which the player can move one of his pawns. You get it.
  • Try to reproduce an existing game you like.
  • Change the rules of an existing game in order to create an interesting variant.
  • Create your own incredible game!
